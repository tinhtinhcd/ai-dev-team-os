## Overview

Linear is a modern project management and issue-tracking platform designed for software teams. It provides a fast, keyboard-first interface for managing issues, cycles, projects, and team workflows.

## Key Concepts

### Navigation Structure
- **Favorites** — Quick access to frequently used views
- **Your teams** — Organize work by team (e.g., Rider app, Driver app)
- **Triage** — Incoming or unassigned issues (badge shows count)
- **Issues** — Main issue list view
- **Cycles** — Time-boxed sprints (Current, Upcoming)
- **Projects** — Grouped initiatives
- **Views** — Custom filtered/saved views

### Issue Structure
Each issue typically includes:
- **ID** — Team prefix + number (e.g., `RIDE-671`, `TIN-1`)
- **Status** — Visual indicator (colored circle: blue = open/in progress, yellow/orange = review/blocked, grey = planned/unstarted)
- **Title** — Short description
- **SLA** — Service Level Agreement tracking (optional)
- **Sub-tasks** — Progress indicators (e.g., `0/3`)

### Workflow
- Issues flow through statuses (e.g., Triage → In Progress → Review → Done)
- Teams can customize workflows per their needs
- Cycles align work to sprints or time periods
